#### 使用教程

1.插件代码拷贝

- 下载后把components目录下mTable/mTable.vue文件拷贝到自己项目目录下


2.插件使用

- vue页面使用

```
<template>
	<view>
		<mTable />
	</view>	
</template>
```


##### 兼容性
uni-app项目中使用都兼容

##### 注意
uniapp不支持命名为table的自定义组件、table是保留关键字
